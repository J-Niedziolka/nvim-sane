# `nvim`-sane

## Installation

1. make sure `vim-gtk3` and `xclip` is installed.
2. run `install.sh` to install Neovim itself.
3. open `lua/plugins.lua` with Neovim. Ignore the errors and warnings,
   run `:so` and then `:PackerSync`.
